## メモ
- BE側で visibility=private / unlisted の資産は `url` 系フィールドに署名付きURLを返す。フロントは TTL 監視・再フェッチを担保する。
- 認証フローは APIキー＋CSRF トークンの二段構成。CORS 環境のためブラウザCookie送信は行わない。
